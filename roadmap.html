Road Map

The main input for Appcceleration is the feedback from the real world projects we work in.

These are the features we're up to in the future:

  • AsyncModule: Rewrite to make it easier to use and better testable, once .Net 4.5 is released.
  • Bootstrapper: client/server extension support
  • Bootstrapper: serviceinstaller
  • Bootstrapper: console argument support
  • EventBroker: support simplified handler method signatures (no need for sender or event args if they are not used).
  • Feature, license and user permission management over the evaluation engine
  • UI navigation for WPF and Web (URL based)
