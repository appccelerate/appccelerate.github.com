StyleCop

StyleCop checks compliance with the coding style guidelines of Appccelerate. Every violation results in an compilation error.

If you want real-time feedback about violations, you can use Resharper with the StyleCop plug-in.

Sanity Checks

There are MSBuild tasks that check for the following:

  • Hint Paths: all hint paths in the project file point to a correct location. (see Check Hint Path Task for more details)
  • Nuget Dependency Versions: checks that the versions of project references and Nuget dependencies match. (see Check Nuget Dependencies Task for more details)

Unit Tests and Specifications

All unit tests and specifications are executed and a failing unit test or specification breaks the build.
